日程概要

State of the Map 2017 は2017年8月18日から20日にかけて福島県会津若松市で開催されます。会津若松市文化センターでお会いしましょう。

私たちは土曜日の午後6時から8時にソーシャルイベントを「ツルカン」こと鶴ヶ城会館ホールで開催します。今年のOSM Awardsはこのソーシャルイベントで発表されます。

  • 8月18日 金曜日: State of the Map の初日です。20分セッションやライトニングトークが予定されています。詳細はタイムテーブルをご覧ください。
  • 8月19日 土曜日: State of the Map 二日目は20分セッションに加えて OSM アワードの発表が予定されています。予定されているセッションはタイムテーブルでご確認ください。
  • 8月20日 日曜日: State of the Map 最終日はマッピングやドキュメント作成などのワークショップが予定されています。一緒に活動することで新しい仲間や知識を発見しましょう。応募の多かったライトニングトークはこの日にも予定されています。

オプショナルフォト

鶴ヶ城まで来た記念にオプショナルフォトはいかがですか? 土曜日の開場前に鶴ヶ城を背景としたグループ写真を撮影します。この撮影をするエリアは通常立ち入り禁止ですが、特別に許可を得て入ることができるようになりました。

オプショナルツアー

日曜日の午後3時45分から福島県立博物館でのオプショナルツアーが行われます。福島の歴史と文化に触れてみましょう。

ブレイクアウトセッション

State of the Map ではブレイクアウトセッション「Birds of a feather」のために使える部屋をいくつか用意しています。

Birds of a feather のために用意した部屋には、セッションについて議論の続きや親睦を深める話をするためのフリーなオープンスペースと、グループで作業をしたりOpenStreetMap関連ソフトウェアのコーディングをしたり、ドキュメントやガイドの作成などのために予約可能な部屋があります。

これらの部屋を使ってやってみたい事はありますか? あなたのアイデアをOpenStreetMapコミュニティに共有する方法はこちらのブログ記事をご覧ください。

観光にも興味があれば計画中の福島県立博物館のツアーにもご期待ください。
